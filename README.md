# 🍊 제주어-표준어 번역기 (Jeju-Standard Translator)

제주어와 표준어 간의 양방향 번역 및 제주어 학습을 위한 퀴즈 기능을 제공하는 웹 서비스입니다.

## ✨ 주요 기능 (Features)

- **양방향 번역 (Bidirectional Translation)**
  - 제주어 ➡️ 표준어 번역
  - 표준어 ➡️ 제주어 번역
- **제주어 퀴즈 (Jeju Dialect Quiz)**
  - 사용자가 재미있게 제주어를 학습할 수 있도록 퀴즈 기능을 제공합니다.
